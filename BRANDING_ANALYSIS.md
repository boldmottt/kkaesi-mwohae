### 🎯 브랜드 강화 제언

#### 1. 로고/아이콘 디자인 방향성
- **컨셉**: "아기 + 시계/태엽 + 말풍선"의 조합
- **색상**: amber-400 (#fbbf24) 메인, amber-600 (#d97706) 텍스트
- **스타일**: 둥근 모서리, flat design (현재 UI와 일관성)
- **산출물**: favicon.ico, app store icon (1024×1024), splash screen

#### 2. 컬러 시스템 정립 (`globals.css` 개선)
```css
/* globals.css에 추가 제안 */
:root {
  --color-brand-primary: #f59e0b;    /* amber-500 */
  --color-brand-secondary: #d97706;   /* amber-600 */
  --color-brand-accent: #fbbf24;      /* amber-400 */
  --color-brand-bg: #fffbeb;          /* amber-50 */
  --color-success: #22c55e;           /* green-500 (기존 유지) */
  --color-info: #a855f7;              /* violet-500 (인사이트용 보조색) */
}
```

#### 3. 브랜드 톤앤매너 가이드라인
- **어조**: 따뜻하지만 전문적인 "육아 전문가 친구" 톤
- **문체**: 존댓말, "~해요/하세요" 종결어미 (현재 UI와 일치)
- **에러 메시지**: "죄송해요, 잠시 후 다시 시도해 주세요." (현재 UI와 일치)
- **CTA 문구**: "시작하기", "저장", "전송" (현재 UI와 일치)

#### 4. 온보딩 경험 개선 (`src/app/onboarding/page.tsx`)
```
[Step 1] 브랜드 소개 화면
  - 로고 + 슬로건 "아기의 깨어있는 시간, 뭐하면 좋을까?"
  - "AI가 아기에 맞춰 매번 새로운 놀이를 추천해요"

[Step 2] 아기 정보 입력 (현재 유지)
  - 이름, 생년월일

[Step 3] 깨시 설정 (현재 유지)
  - 수면 패턴에 따른 자동 계산

[Step 4] 첫 스케줄 생성 (신규)
  - "오늘의 깨시" 자동 생성 + AI 채팅 시작 유도
```

#### 5. 앱 스토어 최적화 (ASO)
- **앱 이름**: 깨시뭐해 - AI 육아 스케줄
- **설명 키워드**: "수면 기록", "영유아 발달", "활동 추천", "AI 육아"
- **스크린샷**: 메인 화면, 추억 페이지, 설정 페이지 5장
